##ABM_Project 

This repository holds the development, milestone and doc files for the development of the Agent-Based-Simulator (ABM) project developed by Pedro Osório Silva and Professor Paulo Garrido from University of Minho, in Julia language.   
Ultimately, this project aims to perform simluations over macro-economy data to allow judgments and conclusions to be taken in what can be changed in order to improve the health state of an economy, mainly, a country.   

###Directories   
* src - Source files with code of the simulator   
* configs - Configuration files for the simulations   
* outputs - Output files of the simulations      
* docs - Documents      
* tools - Tools (Visualizer)
